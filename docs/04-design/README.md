Design system, wireframes, and responsive strategy for the MVP.

## TL;DR

**Design Philosophy:** Clean, accessible, mobile-first.

**Key Tokens:**
- Color: Brand red (#FF6B6B), semantic colors, neutral grays
- Typography: System fonts, 11px-28px scale
- Spacing: 4px base unit
- Layout: 3 breakpoints (mobile, tablet, desktop)

**Responsive Design:**
- Mobile (<576px): Stacked (map 70%, list 30%)
- Tablet (576-1023px): Overlay panels (map full, list drawer)
- Desktop (1024px+): Split (map 60%, list 40% sidebar)

**Accessibility:** WCAG AA color contrast, 44px touch targets on mobile.
